# 게임 에코서버 성능 7일 테스트 : 2월 3일 까지


가동 후 7일차가 되는 날 서버와 더미의 콘솔창의 스크린샷을 보내주세요.


# 게임 에코서버 성능 테스트 조건

2U 에 게임서버
1U 에 더미 x 2

조건 Recv 150만 TPS
조건 Send 150만 TPS
조건 메모리 사용량 1G 미만

총 300만 TPS 이상 목표

더미 설정은 위의 TPS 가 나오도록 적절히 셋팅


위 테스트 진행된 서버를 그대로 재접속 테스트로 전환 테스트를 진행합니다.
성능 테스트를 위한 코드, 재접속 테스트를 위한 코드 변경은 불가합니다.

# 게임 에코서버 재접속 7일 테스트 : 2월 17일 까지

게임 에코서버 재접속 테스트는 위의 성능테스트 직후 바로 진행합니다.
별도의 셋팅 기간이 없으며 성능 테스트에서 진행했던 실행파일을 그대로
재실행 시켜서 재접속 더미로 테스트를 진행 합니다.

재접속 테스트를 위해 빌드를 다시 하거나 다른 실행파일,설정으로 변경은 불가능 합니다.
